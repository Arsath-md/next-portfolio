# 💼 Mohamed Arsath - Developer Portfolio

A modern, responsive developer portfolio built with **Next.js** showcasing my projects, skills, experience, and contact information. The portfolio features a terminal-inspired UI with smooth interactions and a clean design focused on performance and accessibility.

## 🌐 Live Demo

🔗 https://mdarsath-next-portfolio.vercel.app

---

## ✨ Features

- ⚡ Built with Next.js App Router
- 📱 Fully responsive design
- 🎨 Terminal-inspired modern UI
- 🧑 About section
- 🛠️ Skills showcase
- 🚀 Featured projects
- 📄 Resume download
- 📧 Contact form with backend API
- 🔗 Social media integration
- 🌙 Fast page loading and optimized assets

---

## 🛠️ Tech Stack

### Frontend

- Next.js

- JavaScript

### UI & Icons

- React Icons

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Arsath-md/your-portfolio.git
```

Navigate to the project

```bash
cd your-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
